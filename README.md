# BookStore

This  is a simple Book Store web application that allows users to perform CRUD operations on a collection of books. The application is built using React for the frontend and Express with MongoDB for the backend.

## Features

- View a list of books
- View details of a specific book
- Add a new book
- Update an existing book
- Delete a book

## Technologies Used

### Frontend

- React
- React Router
- Axios
- Bootstrap
- React Toastify

### Backend

- Express
- MongoDB
- Mongoose
- CORS

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository:

```sh
git clone https://github.com/yourusername/bookstore.git
cd bookstore
```
2. Install dependencies for the client:
```sh
cd client
npm install
```
3. Install dependencies for the server:
```sh
cd ../server
npm install
```

### Run both Backend and Frontend Server
For Client:
```sh
cd client
npm run dev
```
For Server:

```sh
cd server
nodemon app.js
```
